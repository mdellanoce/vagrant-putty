# What is this?

A [vagrant](http://vagrantup.com) plugin that allows you to [PuTTY](http://www.chiark.greenend.org.uk/~sgtatham/putty/) into a VM.

# How do I use it?

Install it:
<pre>
gem install vagrant-putty
</pre>

Use it:
<pre>
vagrant putty
</pre>

# Assumptions

The putty executable is somewhere on your path.
